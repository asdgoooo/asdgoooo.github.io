  1. 使用Pelican搭建个人blog-1

    四 28 七月 2016 | comments

    WITF

    一直以来都希望有自己的个人博客，随性发点信息，记录自己的生活，写点技术感想，不积跬步无以至千里，不再纸上谈兵，从现在开始。

    Knowledge

    本博客是在mac下搭建完成。搭建博客的工具选用了基于python的pelican，相比wordpress等其它工具来说，它比较轻盈并有很多令人兴奋的特性，再配合免费无限制的github pages，近乎完美。搭建过程中涉及如下技术知识，不过你不必害怕，只是使用它们的开源框架而已，并不需要自己编码，点击可以了解它们是如何的强大，当然你也可以略过它们，后面遇到时再进行了解。

    需要的工具
* gitbub
* python
* pelican
* markdown

    安装
pip install pelican markdown

    生成Site

    在你的电脑上，建立一个目录，作为博客的主目录。我们假定，它的名称为blog。

    mkdir blog
cd blog
pelican-quickstart

    运行一下命令后，就生成了一个站点，包含以下目录：

    blog/
├── content              # 存放输入的源文件 ...
